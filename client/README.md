# Client (React + Vite)

This is the frontend for `amharic-ai-bot`.

## Configure environment

Copy the template and edit as needed:

```bash
cp env.example .env.local
```

- `VITE_API_URL`: backend base URL (default used in code: `http://localhost:3001`)

## Run locally

```bash
npm install
npm run dev
```

Vite runs on `http://localhost:5173` by default.

## Build

```bash
npm run build
```

Output goes to `client/dist`.
